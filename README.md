Axiom-CP 原创作品许可协议

协议简介

Axiom-CP（Axiom-Creator Protection）是一份专为保护原创者权利而设计的公共许可协议。该协议旨在为创作者提供一个清晰、确定的权利保护框架，让创作者能够更安心地进行创作，同时保障公众在合理范围内的使用权利。

核心特点

 强保护性

· 永久性内容禁运：即使原发布平台关闭，协议保护依然有效
· 事前同意原则：任何搬运行为必须获得创作者明确书面同意
· 默示禁止规则：未回复的授权请求一律视为不同意

 严格禁止行为

· 禁止商业使用：未经许可不得以任何形式盈利
· 禁止擅自搬运：包括全文复制、采集存储、文件分发等
· 禁止演绎创作：包括翻译、改编、改写等需要授权

 合理使用空间

· 保障评论与批评自由
· 允许合理引用（符合著作权法规定）
· 支持个人离线阅读使用

如何使用

对于创作者

在您的作品中添加以下声明：

```
本作品采用 Axiom-CP 原创作品许可协议进行保护
```

对于使用者

· 阅读作品即视为接受协议条款
· 需要搬运时，必须联系创作者获得书面授权
· 引用时需控制在合理范围内并明确标注出处

协议版本

当前版本：1.3
生效日期：2025年11月1日

重要条款摘要

关键定义

· 作品：包括文字、图像、图表、代码、音频、视频等原创内容
· 搬运：复制、采集、存储、转载、分发等行为
· 抄袭：内容实质性相似且虚假署名的行为
· 盈利：直接或间接获取经济利益的行为

核心规范

1. 永久保护：不因平台关闭而失效
2. 书面授权：搬运前必须获得明确同意
3. 归属标注：必须注明创作者和原始链接
4. 内容完整：不得篡改原文内容

合理使用

允许在以下情况下使用作品片段：

· 评论与批评
· 学术研究引用
· 个人离线阅读

社区所有权

Axiom-CP协议由创作者社区共同所有和维护：

· 任何创作者都可以免费使用
· 协议文本可供引用，但不得用于商业目的
· 最终解释权归创作者社区

法律依据

本协议基于《中华人民共和国著作权法》及相关国际著作权公约制定，充分尊重并保障法律框架内的合理使用制度。
